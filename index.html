// 01 — System Modules

PROJECT
MODULES

MODULE 01
👤
User Management
Secure registration and login with Firebase Authentication. Profile management and role-based access control to protect sensitive data.
MODULE 02
📈
Stock Monitoring
Fetches live stock prices via Market API. Displays real-time trends and continuously refreshes prices so investors never miss a movement.
MODULE 03
🔔
Alert Management
Set custom target prices for any stock. Triggers instant alerts when price conditions are met — supports Email, SMS, and Push notifications.
MODULE 04
🗄️
Database Management
Firebase Realtime Database stores user details, watchlists, and complete alert history using relational keys for fast lookups.
MODULE 05
📡
Notification System
Multi-channel notification delivery — Email, SMS, and in-app — ensures timely updates for both price increases and decreases.
